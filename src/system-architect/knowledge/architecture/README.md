## 使用指南

1. **按顺序学习**：每天对应一个文件，跟随 5-month-plan 的时间安排
2. **先阅读，再练习**：每个文件末尾有章节练习，务必完成
3. **善用检查清单**：每个文件末尾有学习检查清单，逐项确认
4. **错题回看**：练习中的错题标记出来，第 6 周摸底前集中回顾
5. **周末综合**：周六对照 `04` 和 `08` 做整周回顾
