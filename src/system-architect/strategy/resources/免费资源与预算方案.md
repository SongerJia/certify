## 避坑提醒

1. **不要囤积资源** —— GitHub 下一堆 PDF 不等于学了，选定 2~3 个核心资源吃透
2. **不要只看视频不动手** —— 视频是被动输入，刷题和写论文才是主动输出
3. **旧版教材别用** —— 2020 年前出版的教材不包含云原生/AI/区块链考点
4. **不要过度依赖现成思维导图** —— 自己画的才有记忆效果
5. **不要等"准备好"再刷真题** —— 基础阶段结束后立刻开始，边刷边补
6. **报名后别忘了缴费** —— 每年都有人错过缴费窗口
