## 刷题平台

### 在线平台对比

| 平台 | 核心功能 | 免费/付费 | 推荐指数 |
|:---|:---|:---|:---|
| **[希赛网](https://www.educity.cn/)** | 历年真题 + 章节练习 + 模拟考试 + 错题本 | 部分免费，VIP 全解锁 | ★★★★★ |
| **软考通 APP** | 手机刷题，碎片时间高效利用 | 基础免费 | ★★★★☆ |
| **软考题库（微信小程序）** | 微信内直接刷题，无需下载安装 | 免费 | ★★★★☆ |
| **[ruankao.app](https://ruankao.app/)** | 开源刷题平台，社区维护 | 免费 | ★★★☆☆ |
| **51CTO 题库** | 与课程绑定，购课学员专享 | 购课后免费 | ★★★★☆ |

### GitHub 开源仓库

| 资源 | 说明 |
|:---|:---|
| **[system_architect](https://github.com/xiaomabenten/system_architect)** | 全网最全面的免费备考仓库，含 2009~2024 真题 + 解析 + 论文范文 + 思维导图，配套免费刷题软件 |
| **[软考通电子版](https://github.com/ruankao)** | 社区整理的真题 PDF + 背诵笔记 |

> GitHub 仓库目录结构：
> ```
> 01、教材 + 教程
> 02、视频教程
> 03、历年真题（2009-2024）+ 答案解析
> 04、论文（真题范文 + 写作指导）
> 05、思维导图 + 笔记
> 06、刷题软件
> ```

## 备考工具链

### 笔记与知识管理

| 工具 | 推荐理由 |
|:---|:---|
| **Obsidian** | 本地 Markdown，双向链接 + 图谱视图，构建知识网络 |
| **Notion** | 云端笔记，表格视图管理错题，进度追踪 |
| **Typora** | 论文写作练习，所见即所得 |
| **飞书文档** | 国内访问流畅，支持嵌入导图 |

### 画图工具（案例 + 论文必备）

| 工具 | 用途 |
|:---|:---|
| **Draw.io**（推荐） | 微服务架构图、部署图、ER 图，免费开源 |
| **PlantUML** | 文本绘图，版本可控，适合程序员 |
| **Excalidraw** | 手绘风格白板，快速草图 |

### 刷题辅助

| 工具 | 用途 |
|:---|:---|
| **软考通 APP** | 通勤/排队碎片时间刷选择题 |
| **Anki** | 自制闪卡，间隔重复记忆（法规/概念类） |
| **Excel / Notion** | 错题本管理，按模块/错误原因分类统计 |
